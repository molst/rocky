## Artifacts

[on Clojars](https://clojars.org/rocky). If you are using Maven, add the following repository
definition to your `pom.xml`:

``` xml
<repository>
  <id>clojars.org</id>
  <url>http://clojars.org/repo</url>
</repository>
```

With Leiningen:
```
[rocky "0.1"]
```
